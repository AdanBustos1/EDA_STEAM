## 📝 Descripción del Proyecto

Este proyecto realiza un **Análisis Exploratorio de Datos (EDA)** completo sobre el catálogo de videojuegos disponibles en la plataforma **STEAM**. El objetivo principal es identificar patrones, tendencias y variables clave que influyen en el éxito de los juegos, proporcionando recomendaciones estratégicas para desarrolladores de videojuegos de diferentes escalas.

El análisis está orientado a responder preguntas críticas como:
- ¿Qué características definen a un juego exitoso en STEAM?
- ¿Cuál es el precio óptimo según el género y tipo de desenvolvimiento?
- ¿Qué géneros tienen mayor demanda y rentabilidad?
- ¿Cómo se distribuyen los recursos según la escala del proyecto?

---

## 🎯 Estrategias por Tipo de Desarrollador

El proyecto identifica **3 estrategias principales** basadas en los recursos disponibles:

| Tipo | Recursos | Enfoque |
|------|----------|---------|
| **Indie** | Bajos | Nichos específicos, innovación, márgenes altos |
| **AA (Doble A)** | Limitados | Balance entre presupuesto y alcance, géneros populares |
| **AAA (Triple A)** | Grandes | Producciones de alto presupuesto, máxima visibilidad |

## 📊 Fuentes de Datos

El análisis utiliza tres datasets principales:

- **steam.csv**: Información detallada de juegos (título, fecha de lanzamiento, precio, reviews, etc.)
- **steam_requirements_data.csv**: Requisitos técnicos mínimos y recomendados
- **steamspy_tag_data.csv**: Etiquetas y categorización de juegos

Ubicados en: `/Data/`

## 📈 Contenido del Análisis

El notebook incluye:

1. **Carga y Limpieza de Datos**: Tratamiento de valores faltantes y anomalías
